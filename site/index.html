What it does

Secure Review Webhooks

HMAC signature verification + replay protection for GitHub webhook events.

Incremental Re-review

@openpr review tracks reviewed commit state and only re-analyzes deltas.

Safe Autofix

@openpr fix applies deterministic low-risk changes with policy gates and dry-run support.

Model Flexibility

Use Mock, OpenAI, Anthropic, or local Ollama adapters with timeout/retry controls.
