AI PR reviewer + safe autofix bot for GitHub.

Why teams use OpenPR

  • Secure webhook verification + replay protection
  • Incremental @openpr review with commit-aware state
  • Safe deterministic @openpr fix with dry-run mode
  • Model adapters: Mock, OpenAI, Anthropic, Local (Ollama)

Quick start

cp .env.example .env
npm install
npm run lint && npm run typecheck && npm test
npm run dev
